# TextCollection Bundle Editor (TCBE)

![Version](https://img.shields.io/badge/version-2.2.0-blue)
![Platform](https://img.shields.io/badge/platform-Windows-success)
![Framework](https://img.shields.io/badge/.NET-8.0-purple)
![Unity](https://img.shields.io/badge/Unity-AssetBundles-black)

Professional Unity Localization Tool

O **TextCollection Bundle Editor** é uma ferramenta criada para editar arquivos **Unity AssetBundle** que utilizam a estrutura **TextCollection** e **LocalisedText**, permitindo traduzir jogos diretamente sem recompilar AssetBundles manualmente.

---

# Recursos

- ✅ Abrir arquivos `.bundle`
- ✅ Detectar automaticamente TextCollections
- ✅ Editar traduções diretamente
- ✅ Salvar novamente no Bundle
- ✅ Interface V1 (Tabela)
- ✅ Interface V2 (Editor)
- ✅ Pesquisa por texto ou ID
- ✅ Filtros
- ✅ Exportar CSV
- ✅ Importar CSV
- ✅ Backup automático
- ✅ Validação de placeholders
- ✅ Tema escuro

# Como usar

1. Abra o programa.

2. Clique em **Abrir Bundle**

3. Escolha uma coleção.

4. Traduza os textos.

5. Clique em **Salvar**.
